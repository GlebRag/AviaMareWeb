Decoding of commit changes descriptions marks

There symbols and marks that are used to shorten some change descriptions. With these marks you can specify/find out what specific change did.

Mark Meaning
HF Hotfix
F Fix
+ New feature
- Removed
> Edited/Renamed
>> Moved
! Important change
M Git branch merge
No mark Unclassified change
